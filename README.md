# JBKnowledge React Development

## Installation

```bash
npm install @jbknowledge/eslint-config
npm install @jbknowledge/react-dev
```

**package.json**
```json
{
  "scripts": {
    "lint": "jbk-scripts lint .",
    "start": "jbk-scripts start"
  },
  "eslintConfig": {
    "extends": ["@jbknowledge"]
  }
}
```
